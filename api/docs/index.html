📘 Bienvenue sur la documentation de l’API Biscord

Cette documentation couvre tous les endpoints REST disponibles sur https://biscord-api-stg.xcsoftworks.com/api/

🔐 Authentification

Toutes les routes nécessitent une session PHP active côté serveur. Il n’y a pas encore de token d’API ou d’OAuth public.

📦 Format des données

Les requêtes et réponses sont au format JSON. Exemple :

{
  "success": true,
  "data": {...}
}
