🔗 accept_invite.php – POST

Permet à un utilisateur connecté d'accepter une invitation à un serveur à l'aide d'un code.

📤 Paramètres requis (body POST)

{
  "code": "abc123"
}

📥 Réponse – Succès

{
  "success": true,
  "server_id": 42
}

❌ Réponse – Erreur : données manquantes

{
  "success": false,
  "error": "Données manquantes."
}

❌ Réponse – Erreur : code invalide

{
  "success": false,
  "error": "Invitation invalide."
}
